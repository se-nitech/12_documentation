## コード

コードを入力する場合は以下のようなコードブロックを使います．

```python:test.py
def myfunc(a, b, c):
    a = b + c
    print(a)
```

## 数式

レンダラによっては$y = f(x)$のようなLaTeX数式も対応しています．インラインだけでなく，行立ての数式
$$
y = \int_a^b f(x, t) dt
$$
も対応しています．数式番号は対応していないことがほとんどです（拡張機能が使える場合もある）．
